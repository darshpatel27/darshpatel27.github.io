Book Recommendations

Fri, Jan 21, 2022

The following books have really helped me in getting better at different competencies and i would highly recommend it to all

Tech

  1. The Design Of The Unix Operating System - Maurice J Bach
  2. Designing Data-Intensive Applications - Martin Kleppmann
  3. Introduction to Algorithms - CLRS
  4. Refactoring: Improving the Design of Existing Code - Martin Fowler
  5. Design Patterns: Elements of Reusable Object-Oriented Software - Erich Gamma
  6. Clean Code: A Handbook of Agile Software Craftsmanship - Robert C Martin
  7. The Effective Engineer - Edmond Lau

Start-ups

  1. The hard thing about hard things - Ben Horowitz
  2. Zero to One - Peter Thiel

Leadership

  1. High Output Management - Andrew S Grove
  2. Leading - Sir Alex Ferguson
